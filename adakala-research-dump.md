- **Indonesia** (BGG 19777) — designed by Dutch designers (Jeroen Doumen/Joris Wiersinga), published by Splotter Spellen (Netherlands). Indonesian *theme* only, not Indonesian-made.
- **Buru** (BGG 316312) — designed by Stephen Selego/Alex Flagg, published by Crafty Games (US). Indonesian studio Ketchup Games only served as cultural consultants, not designer/publisher of record.

### Catatan teknis scraping (metodologi)

- Akses langsung ke `boardgamegeek.com` via `curl`/`WebFetch` diblokir Cloudflare (403/401, termasuk endpoint XML API resmi `xmlapi2` yang biasanya publik — kemungkinan IP environment ini kena blok umum).
- Solusinya: pakai browser asli (`claude-in-chrome`) yang berhasil melewati Cloudflare challenge secara otomatis, lalu ambil data terstruktur langsung dari objek JS internal BGG: `window.GEEK.geekitemPreload.item` (berisi `name, images, stats.average, stats.avgweight, rankinfo, links.boardgamecategory, links.boardgamemechanic, links.boardgamepublisher, links.boardgamedesigner`, dll). Ini jauh lebih akurat & lengkap dibanding scraping HTML biasa.
- Field `rank` yang dipakai adalah `rankinfo` dengan `rankobjecttype:"subtype"` dan `browsesubtype:"boardgame"` (rank keseluruhan BGG, bukan rank per-family/kategori seperti "Family Game Rank").
- Gambar cover diunduh langsung dari CDN `cf.geekdo-images.com` (tidak kena blok Cloudflare seperti domain utama), disimpan ke `bgg-database/images/{id}.{ext}` sesuai ekstensi asli dari URL BGG (ada yang `.png`, `.jpg`, bahkan satu `.gif` untuk Waroong Wars: Second Edition).
